# WordMaster

> 基于艾宾浩斯遗忘曲线的 Android 背单词 App
> Kotlin + Jetpack Compose + Room

## 作者
- xhmTmax530 (13868078255@139.com)

## 构建
```bash
ANDROID_HOME=$HOME/.android_sdk ./gradlew :app:assembleDebug
```

## 装到手机
```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```
